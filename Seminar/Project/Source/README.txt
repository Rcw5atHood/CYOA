Travel Adventure Game

Project Overview:
The Travel Adventure Game is a text-based adventure game developed in Python using the Pygame library. It allows players to explore various destinations and play different games at each location.

Repository Structure:
The project repository has the following directory structure:
- main.py: The main Python script that runs the game.
- connect4.py: Module for the Connect 4 game.
- tictactoe.py: Module for the Tic Tac Toe game.
- tetris.py: Module for the Tetris game.
- constants.py: File containing constant values used in the project.
- assets/: Directory containing game assets such as images and soundtracks.

Tech Stack:
The Travel Adventure Game utilizes the following software tech stack:
- Python: Programming language used for game development.
- Pygame: Python library for creating games and multimedia applications.
